# Flipping Card UI Design in HTML & CSS | Card Hover Animation | ASMR programming

Um projeto simples de **card com efeito flip** usando HTML e CSS.  
Ao passar o mouse, o card gira revelando o conteúdo do verso.

## ✨ Sobre

Este projeto demonstra como criar uma animação de **flip (giro)** em um card utilizando apenas CSS.

## 🛠️ Tecnologias

- HTML
- CSS (transform, transition)

## 🎯 Funcionalidades

- Efeito de hover com animação
- Flip 3D
- Design simples e moderno

1. Baixe ou clone o repositório
